== Description ==

The WP AI Summarizer Plugin is a powerful tool that leverages OpenAI to summarize post content and product reviews on your WordPress website. It provides an easy way to generate concise summaries of your content and display them in a meta box in the admin panel. Whether you want to summarize blog posts or product reviews, this plugin has you covered.

== Installation ==

1. Upload the "wp-ai-summarizer" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to the plugin settings page located in the WordPress dashboard.
4. Configure the plugin options, such as setting your OpenAI API key and CRON delay time.
5. Save the settings.
6. Now, you can use the plugin to generate summaries for your posts and reviews.

== Admin Settings Fields ==

The plugin provides the following configuration options in the admin settings:

- OpenAI API Key: Enter your OpenAI API key to enable summarization.
- CRON delay time (default: 300 seconds): Set the delay time of the CRON.

== Frequently Asked Questions ==

Q: How does the Custom Summarizer Plugin work?
A: The plugin utilizes OpenAI to create concise summaries of your content and displays them in a meta box in the admin panel.

Q: Do I need an OpenAI API key?
A: Yes, you must provide your OpenAI API key in the settings to enable summarization.
